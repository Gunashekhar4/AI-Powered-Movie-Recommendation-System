# Running the System

## Instructions to Run the Movie-Recommendation-System.py

1. **Open your Terminal.**  
   - Make sure you are in the directory where the `Movie-Recommendation-System.py` file is located.

2. **Ensure Python is installed.**  
   - You can check if Python is installed by typing `python --version` or `python3 --version` in your terminal.
   - If it is not installed, download and install it from [python.org](https://www.python.org/downloads/).

3. **Install the required libraries.**  
   - Before running the script, make sure to install the necessary libraries using pip. Run the following command:
     ```bash
     pip install -r requirements.txt
     ```

4. **Run the script.**  
   - Once you have all the dependencies installed, you can run the script by executing the following command in your terminal:
     ```bash
     python Movie-Recommendation-System.py
     ```

5. **Provide input as required.**  
   - Follow the on-screen prompts to input any required data, such as movie preferences or user ratings.

6. **Check the output.**  
   - The recommendations will be displayed in the terminal once the script has finished running.

7. **Experiment with different inputs for better recommendations.**  
   - Feel free to run the script multiple times with different inputs to see varied results.
